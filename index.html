📋 One-time setup (2 mins):

  1. Go to github.com → Sign up (free) or log in
  2. Tap your profile photo → Settings
  3. Scroll down → Developer settings
  4. Personal access tokens → Tokens (classic)
  5. Generate new token (classic)
  6. Name: TLE Sync · Expiry: No expiration
  7. Check only gist ✓ → Generate token
  8. Copy the token (starts with ghp_) → paste below ↓

✅ Use the same token on your phone and desktop — all data syncs automatically.
